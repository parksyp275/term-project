## 1. 프로젝트 소개
최근 IT 채용 시장의 트렌드를 파악하기 위해 만든 프로그램입니다.
해외 채용 사이트에서 실시간으로 데이터를 가져와서, 내가 관심 있는 기술이나 직군을 검색하면 얼마나 많이 뽑는지 그래프로 보여줍니다.

## 2. 주요 기능
* **실시간 공고 확인:** API를 통해 최신 원격 근무 공고를 가져옵니다.
* **키워드 검색:** 'Python', 'Data' 같은 단어로 검색할 수 있습니다.
* **시각화:** 검색된 직군별 비율을 막대 그래프로 그려줍니다.
* **파일 저장:** 검색 결과를 엑셀이나 CSV 파일로 저장해서 나중에 다시 볼 수 있습니다.

## 3. 사용 라이브러리
이 코드를 실행하려면 아래 라이브러리가 필요합니다.
* `requests`: 데이터 통신
* `pandas`: 데이터 정리 및 엑셀 저장
* `matplotlib`: 그래프 그리기
* `openpyxl`: 엑셀 파일 처리
